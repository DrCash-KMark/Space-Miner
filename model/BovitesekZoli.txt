Az aszteroidához felvettem egy napkozelseg flaget.
Javitom a hibasan felvett tagvaltozokat. (Nev es tipus felcserelve.)

Felvennek egy destroyMe(Controllable) metodust a Game-be, hogy az a kor vegen fel tudja szabaditani a megsemmisulo aszteroidakat.

Fel kellett vennem egy asteroidExploded() fuggvenyt az entitasba, hogy a felrobbanaskor az entitas maga tudja eldonteni,
hogy at kell repulnie egy szomszedos aszteroidara (Robot), vagy meg kell halnia(Settler).

Felveszek egy notifyAllAboutSunFlare() metodust a game-be, hogy a Sun ertesithesse a tobbi objektumot.
+ Felveszek egy getNotifiedAboutSunFlare metodust a Controllable interface-be.

Csinalnek egy isHollow() metodust az asteroidahoz, mert segitene tesztelni pl. az elbujas lehetoseget.

A removeInventory-nak visszateresi erteket allitottam be. (Material)

Altalam letrehozott metodusok leirasa:
Game -> destroyMe(toDestroy : Controllable): void	Meghívásával egy Controllable értesíti a Game-et, hogy az szüntesse meg a rá mutató hivatkozást a kör végén.
Game -> notifyAllAboutSunFlare() : void				A Sun hívja a Game-en, amikor napkitörést kezdeményez. Hatására a Game értesíti az összes Controllable-t a getNotifiedAboutSunFlare hívással.

Controllable -> getNotifiedAboutSunFlare() : void	Napkitörésről értesül a Controllable interface-t megvalósító objektum.

Entity -> asteroidExploded(): void 					Ezt a metódust egy felrobbanó aszteroida hívja meg az összes rajta található entitáson, hogy azok tudják, hogy me kell halniuk, vagy szomszédos aszteroidára kell ugraniuk.

Asteroid -> isHollow() : Boolean					Visszatérési értékében megadja, hogy üreges-e az aszteroida.
